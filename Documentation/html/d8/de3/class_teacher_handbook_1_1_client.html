static void TeacherHandbook.Client.LoadXAML_LangDicts ( CultureInfo  targetCulture,
CultureInfo  secondaryCulture 
)
static

Scaning for language dictionaries in XAML files, and load them to Merged dictionaries. Load relative to new culture if found. Leave previous culture if not.

Require files format: *.lang.CULTURE_CODE.xaml, where culture code equal current translation of the app. Example: plugin.feed.lang.en-US.xaml

Parameters
targetCultureCulture that will be serched with hightest priority.
secondaryCultureCulture that will be prefured in case of target not implemented. If also not implemented than will be used first entry.
